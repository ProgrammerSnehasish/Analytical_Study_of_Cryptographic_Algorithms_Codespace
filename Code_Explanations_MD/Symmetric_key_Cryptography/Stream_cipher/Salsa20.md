# Salsa20 Code
```python
from Crypto.Cipher import Salsa20
import os

def salsa20_encrypt(key: bytes, plaintext: str) -> tuple[bytes, bytes]:
    """Encrypt plaintext using Salsa20 and return nonce and ciphertext"""
    cipher = Salsa20.new(key=key)
    ciphertext = cipher.encrypt(plaintext.encode('utf-8'))
    return cipher.nonce, ciphertext

def salsa20_decrypt(key: bytes, nonce: bytes, ciphertext: bytes) -> str:
    """Decrypt ciphertext using Salsa20 and return plaintext"""
    cipher = Salsa20.new(key=key, nonce=nonce)
    decrypted = cipher.decrypt(ciphertext)
    return decrypted.decode('utf-8')

# === User Input ===
key_input = input("Enter a 32-byte (256-bit) key: ")
if len(key_input) != 32:
    raise ValueError("Key must be exactly 32 characters (256 bits).")

plaintext_input = input("Enter the plaintext to encrypt: ")

# Convert key to bytes
key = key_input.encode('utf-8')

# Encrypt
nonce, ciphertext = salsa20_encrypt(key, plaintext_input)
print("\nNonce (hex):", nonce.hex())
print("Encrypted (hex):", ciphertext.hex())

# Decrypt
decrypted = salsa20_decrypt(key, nonce, ciphertext)
print("\nDecrypted plaintext:", decrypted)
```
## 🧩 Algorithmic Approach (Step-by-Step)

This program uses the Salsa20 stream cipher (invented by Daniel J. Bernstein) to encrypt and decrypt text using a 256-bit (32-byte) key and an automatically generated nonce (Initialization Vector).

### 🔹 Step 1 — Input & Key Preparation

Prompt the user for a 32-character key (256 bits).

Validate key length (must be exactly 32 bytes).

Convert key from string → UTF-8 encoded bytes.

### 🔹 Step 2 — Encryption Process

Initialize Salsa20 cipher:
```python
cipher = Salsa20.new(key=key)
```
Internally, this generates a random 64-bit nonce (8 bytes).

Encrypt plaintext:
```python
ciphertext = cipher.encrypt(plaintext.encode('utf-8'))
```
The plaintext is first encoded to bytes.
The Salsa20 keystream (dependent on key + nonce) is XORed with the plaintext bytes.
Return the nonce and ciphertext.

### 🔹 Step 3 — Decryption Process

Recreate the cipher object with the same key and nonce:
```python
cipher = Salsa20.new(key=key, nonce=nonce)
```
Decrypt the ciphertext by XORing it with the same keystream:
decrypted = cipher.decrypt(ciphertext)
Convert decrypted bytes → readable text using UTF-8.

### 🔹 Step 4 — Output
Print:
Nonce (in hex format)
Ciphertext (in hex format)
Decrypted plaintext (original message)

|Step|	Operation|	Details|	Complexity|

|2|	Cipher Initialization|	Create Salsa20 cipher with random nonce|	O(1)|
|3|	Encryption|	XOR plaintext bytes with keystream|	O(n)|
|4|	Decryption	XOR ciphertext| bytes with keystream|	O(n)|

|Operation|	Time Complexity|	Space Complexity|	Explanation|

|Keystream Generation|	O(n)|	O(1)|	Keystream generated on-the-fly for each byte.|
|XOR Encryption/Decryption|	O(n)|	O(1)|	One XOR per byte.|
|Input/Output Conversion|	O(n)|	O(n)|	String ↔ Bytes conversion and hex formatting.|
|Overall|	O(n)|	O(n)|	Linear in plaintext length.|

## 🔐 Key Notes

- Salsa20 is a **stream cipher**.

- The nonce must never be reused with the same key — doing so repeats keystreams and breaks security.

- This implementation uses a secure random nonce generated by the library.

- Salsa20 offers very high performance and excellent resistance to timing attacks.
